# TOUGHTS

Aplicação web desenvolvida com Node.js, Express, Sequelize e MySQL.
Inclui sistema de autenticação, CRUD de pensamentos, views com template engine, e organização seguindo o padrão MVC.

## Tecnologias utilizadas

Node.js,
Express,
MySQL + Sequelize (ORM)
Template Engine, 
MVC Architecture,
Sessions para autenticação

### Funcionalidades

Cadastro e login de usuários
Publicação, edição e exclusão de pensamentos
Busca e ordenação de posts
Projeto desenvolvido como parte de um curso de Node.js e banco de dados relacionais
