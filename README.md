# 💭 Thoughts

Aplicação web desenvolvida com Node.js, Express, Sequelize e MySQL. Inclui sistema de autenticação, CRUD de pensamentos, views com template engine, e organização seguindo o padrão MVC.

## ⚙️ Funcionalidades

- Cadastro e login de usuários
- Publicação, edição e exclusão de pensamentos
- Busca e ordenação de posts
- Sistema de autenticação com Sessions

## 🚀 Como rodar o projeto

### Pré-requisitos

- [Node.js](https://nodejs.org/)
- [MySQL](https://www.mysql.com/) rodando localmente

### Instalação

```bash
# Clone o repositório
git clone https://github.com/Renan-Lopes7/thoughts-project.git

# Acesse a pasta
cd thoughts-project

# Instale as dependências
npm install
```

### ⚠️ Configuração do banco de dados

Crie um banco de dados MySQL e substitua as credenciais no arquivo de configuração do banco pelos seus dados de acesso ao MySQL.

### Execução

````nodemon
```

## 🛠️ Tecnologias

- **Node.js** — runtime JavaScript
- **Express** — framework web
- **Sequelize** — ORM para banco de dados
- **MySQL** — banco de dados relacional
- **Template Engine** — renderização de views
- **Sessions** — autenticação de usuários
- **Nodemon** — reload automático em desenvolvimento
- **MVC** — arquitetura do projeto
